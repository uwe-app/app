hypertext 0.7.0
Fast and elegant site generator

USAGE:
    ht [FLAGS] [OPTIONS] [ARGS] [SUBCOMMAND]

FLAGS:
        --follow-links      Follow symbolic links
        --force             Disable incremental build
        --help              Prints help information
        --html-extension    Disable clean locations
    -l, --live              Enable live reload
        --loose             Disable strict mode
    -r, --release           Generate a release build
    -V, --version           Prints version information

OPTIONS:
    -d, --directory <directory>    Build input sub-directory
    -h, --host <host>              The name of the host [default: localhost]
        --log-level <log-level>    Log level [default: info]
        --max-depth <max-depth>    Maximum depth for recursion
    -p, --port <port>              The port number [default: 3000]
    -t, --tag <tag>                Build tag name

ARGS:
    <input>     Read files from directory [default: site]
    <output>    Write files to directory [default: build]

SUBCOMMANDS:
    archive    Create zip archive
    build      Compile a site
    bundle     Bundle a site into executables (requires Go)
    help       Prints this message or the help of the given subcommand(s)
    init       Generate site source files
    serve      Serve site files
